# AI應用規劃師題庫與考古題水平分析

## 結論

現行主題庫已改為高擬真模擬題版本。原本第一科、第二科各 1000 題不再只是概念練習，而是改成更接近考古題的短題幹、單一考點、中英術語、相近概念干擾、統計計算與程式碼判讀題型。

目前仍保留雙層題庫設計：原兩科各 1000 題作為主體高擬真模擬題，另有「考古題與高擬真補強」科目，收錄公告試題與額外補強題。

- 解析已高擬真化：解析不只給正解，也補充判斷線索與排除干擾選項的理由，適合考後檢討。

## 仍需注意的部分

- 題庫仍是自動生成題，正式考古題的附圖題、表格輸出截圖題目前只能以文字化情境模擬。
- 第二科程式碼題為文字語法判讀，尚未嵌入真正圖片或執行輸出表格。
- 高擬真題允許選項組合重複，因為正式考題常反覆使用相似語法選項；稽核重點改為題幹唯一、題數配置與考點覆蓋。
